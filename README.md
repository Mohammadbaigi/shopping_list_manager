# Shopping List Manager

A simple command-line shopping list manager built with Python.

## Features
- Add multiple items to a shopping list
- Display the list with numbered items
- Remove an item by selecting its number
- View the updated list after removal

## How it works
1. The program asks how many items you want to add.
2. You enter each item one by one.
3. The list is displayed with numbers.
4. You choose the number of an item to remove.
5. The updated list is shown.

## Concepts practiced
- Lists
- Loops (for)
- User input
- List methods (append, remove)
